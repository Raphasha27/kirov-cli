# Kirov CLI 🦀

A blazing-fast Command Line Interface for managing the Kirov Dynamics AI and Cybersecurity ecosystem. Written purely in Rust for maximum performance and memory safety.

## Installation

```bash
cargo build --release
./target/release/kirov-cli status
```

## Commands
- `status`: Get system-wide funding/API readiness status.
- `sandbox`: Deploy secure zero-container isolated execution.
- `agents`: Trigger autonomous developer agents.
